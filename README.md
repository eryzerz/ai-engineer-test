## Dependencies (notebooks 1 & 2)

Notebooks [1-small-parse.ipynb](1-small-parse.ipynb) and [2-large-parse.ipynb](2-large-parse.ipynb) use **pandas**, **numpy**, and **Jupyter**. Install from this directory using [requirements.txt](requirements.txt):

```bash
python -m venv .venv
```

Activate the virtual environment, then:

```bash
pip install -r requirements.txt
```
